## Cross-Repository Protocols

### TimeIsland Protocol (TI-YYYY-MM-NNN)
Every significant decision creates a TimeIsland:
1. Philosophy-of-AI defines principles → TI generated
2. Genesis-ChainSet0.1 initializes modules → TI recorded
3. ai-soul-spine-system executes steps → TI tracked
4. tonesoul-codex contracts decisions → TI archived
5. tone-soul-integrity validates output → TI finalized
6. governable-ai publishes policies → TI completed
7. Cycle repeats with new insights

### Responsibility Chain Contract (RCC)
Each PR/commit must include:
- TI-ID: Temporal identifier [YYYY-MM-NNN]
- Bounded Context: Which domain
- G-P-A-R Phase: Current node
- SourceTrace: Origin tracking
- ResonanceSignal: {value_fit, consensus, risk}
- Gate Scores: {FS, POAV, SSI, LC}
- Reviewers: Which repos must review

### Knowledge Feedback Loop
- Bidirectional sync: Philosophy ↔ Engineering
- Automatic weight propagation
- Pattern recognition from execution
- Circular accountability
